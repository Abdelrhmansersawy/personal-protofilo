Built Java Swing text editor using Rope (Treap) for O(log n) edits; handled text files up to 50MB.

Implemented undo/redo with Command pattern and KMP-based search; improved find/replace speed by 40%.

Integrated Trie-backed spell-checker (100k+ words) with DP edit distance; reduced typo detection time by 30%.
