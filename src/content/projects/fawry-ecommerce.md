Engineered a production-grade e-commerce platform with 8+ Spring Boot microservices as part of a 5-member team during my internship at **Fawry** — Egypt's leading digital transformation and fintech platform.

## Key Contributions

- **API Gateway**: Designed a secure and highly scalable API gateway with JWT authentication and rate-limiting; reduced average response time by 25%.
- **Event-Driven Architecture**: Integrated Kafka-based event-driven architecture for reliable, real-time inter-service communication.
- **Microservices**: Built and maintained 8+ Spring Boot microservices handling core e-commerce functionality.

## Tech Stack

Java, Spring Boot, Kafka, Docker, RESTful APIs, JWT, PostgreSQL.
